# Projet---Dashboard-fictif
Petit projet de dashboard multi-région avec prévion 

📋 Description
Ce projet est un outil de Business Intelligence (BI) destiné à la direction. Il permet de visualiser les performances commerciales historiques et d'anticiper les revenus futurs grâce à l'intelligence artificielle.

Fonctionnalités clés :

Analyse Temporelle : Suivi du Chiffre d'Affaires (CA) par mois et par année.

Segmentation : Visualisation par région et par catégorie de produits via des graphiques interactifs Plotly.

Prévision IA : Intégration d'un modèle Prophet (Meta) pour estimer le CA des 30 prochains jours avec un indice de fiabilité (MAPE).

Filtres Dynamiques : Exploration personnalisée par zone géographique et période.
